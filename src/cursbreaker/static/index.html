No API key? How to get one (it's free to start)

An API key is like a password that lets CursBreaker use Google's Gemini AI on your behalf. You create your own — it takes a couple of minutes and you need only a Google account (the same one you use for Gmail).

  1. Open Google AI Studio and sign in with your Google account.
  2. Click Create API key. If it asks which project to use, let it make a new one — the name doesn't matter.
  3. Copy the key it shows you — a long string that starts with AIza…
  4. Paste it into the box above and click Save key.

Google has a free tier (no credit card required) with daily limits; heavier use needs billing. CursBreaker estimates the cost before every run. Full step-by-step guide & pricing →

      Glossary — what these terms mean (plain language)
      Handwriting transcription
      Turning the handwriting in a scanned page into typed, editable text.
      OCR
      Software that reads text out of an image.
      hOCR
      A file that stores each recognized word and the spot on the page where it sits, so the page becomes searchable.
      ALTO XML
      A library/archive standard file holding the same word positions as hOCR, in a different format.
      Bounding box
      The rectangle marking where a line or word sits on the page.
      API key
      A private code that lets this app use Google's Gemini service on your behalf.
      DPI
      Dots per inch — how finely a PDF page is turned into an image. Higher is sharper but larger.
      Token
      The unit Google counts, and bills, for the text and images it reads and writes.
